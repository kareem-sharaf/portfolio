Spring Boot PostgreSQL WebSocket JWT
Problem: Medical clinic management system requiring secure dual authentication (employees vs patients), real-time availability calculation considering work schedules and holidays, immutable medical records after finalization, WebSocket chat security, and appointment rescheduling with history tracking.
Solution:

  • Work schedule templates with day-of-week patterns, holiday date range checking, and dynamic slot generation
  • isFinalized boolean flag with business logic preventing modifications after record completion
  • WebSocket channel interceptors validating JWT tokens with session-based access control
  • Parent-child booking relationships maintaining rescheduling history
Technologies: Spring Boot 3.2.8, Spring Security, Spring Data JPA, PostgreSQL, Spring WebSocket (STOMP), JWT (jjwt), Firebase Admin SDK, MapStruct (DTO Mapping), Flyway Migrations, Jakarta Bean Validation
Result: Digitized clinic operations reducing manual paperwork. Appointment booking system prevents double-booking and scheduling conflicts. Medical record system maintains comprehensive patient history improving continuity of care. Real-time chat facilitates patient-doctor communication. Dual authentication ensures HIPAA-compliant security standards. Reduced admin time by 60%, improved patient satisfaction by 40%.
